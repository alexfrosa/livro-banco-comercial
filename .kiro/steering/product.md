# Product

Interactive Banking Book is a static digital book that teaches how commercial banks work. It targets learners who want to understand modern banking systems — from KYC and accounts to credit, payments, treasury, ALM, and risk management.

The book is hosted on GitHub Pages with no backend. All content lives in the repository as Markdown/MDX files. Key features:

- Chapter/section navigation with progress tracking (localStorage)
- Interactive simulations (e.g., fractional reserve money creation)
- Glossary panel with inline term highlighting
- Mermaid diagrams and animations
- Light/dark theme with anti-FOUC
- Fully accessible (WCAG 2.1 AA/AAA target, keyboard nav, screen readers)
- Responsive: mobile (< 768px), tablet (768–1024px), desktop (1025px+)

Content is in Brazilian Portuguese. UI labels, ARIA attributes, and copy should follow pt-BR conventions.

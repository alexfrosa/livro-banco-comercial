Este livro nasceu de uma pergunta simples: **por que é tão difícil entender como um banco funciona por dentro?**

Existem muitos materiais sobre finanças pessoais, investimentos e economia. Mas poucos explicam, de forma prática e acessível, o que acontece nos bastidores de um banco comercial — os sistemas, os processos, as regras e as pessoas que fazem tudo funcionar.

## O que você vai aprender

Ao longo deste livro, você vai entender:

- Como um banco é estruturado internamente, da captação ao crédito
- Quais sistemas processam cada tipo de operação
- Como o dinheiro é registrado, movimentado e controlado
- Quais são as obrigações regulatórias e como os bancos as cumprem
- O que acontece quando algo dá errado — e como o backoffice resolve

O foco é o **backoffice bancário**: a parte menos visível, mas mais crítica de qualquer instituição financeira.

## Para quem é este livro

Este guia é para você que:

- Trabalha ou quer trabalhar em tecnologia bancária
- Está desenvolvendo sistemas para o setor financeiro
- Quer entender o domínio antes de escrever código
- É curioso sobre como o sistema financeiro realmente funciona

Não é necessário ter experiência prévia em finanças. Os conceitos são apresentados de forma progressiva, do mais simples ao mais complexo.

## Como usar este guia

O livro é organizado em **partes temáticas**, cada uma cobrindo uma área do banco. Dentro de cada parte, você encontra:

1. **Capítulos de conceitos** — explicam os fundamentos teóricos
2. **Capítulos de jornada** — mostram como os processos funcionam na prática
3. **Capítulos de simulação** — permitem experimentar os conceitos interativamente

Você pode ler linearmente ou pular para a parte que mais interessa. Os capítulos de simulação são especialmente úteis para fixar o aprendizado — use-os depois de ler os conceitos correspondentes.

## Uma nota sobre o contexto brasileiro

Os exemplos e regulações neste livro são baseados no **sistema financeiro brasileiro**, regulado pelo Banco Central do Brasil (BACEN) e pelo Conselho Monetário Nacional (CMN). Muitos conceitos são universais, mas os sistemas de pagamento (PIX, STR, SPI), as obrigações regulatórias (SCR, CCS, COSIF) e as normas citadas são específicos do Brasil.

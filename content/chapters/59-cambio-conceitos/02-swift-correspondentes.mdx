Quando um banco brasileiro precisa enviar dinheiro para o exterior — ou receber de lá — não existe uma conexão direta entre todos os bancos do mundo. O sistema funciona por meio de uma rede de correspondentes bancários, conectados pela mensageria SWIFT.

## SWIFT — Society for Worldwide Interbank Financial Telecommunication

O SWIFT é uma cooperativa de instituições financeiras que opera uma rede segura de mensagens para transferências internacionais. Não movimenta dinheiro diretamente — é um sistema de mensageria que instrui os bancos a movimentar fundos.

**Código SWIFT/BIC:** cada instituição financeira tem um código único de 8 ou 11 caracteres (ex: BRASBRRJXXX para o Banco do Brasil em São Paulo). Esse código identifica o banco, o país, a cidade e a agência.

**Mensagem MT103:** é o tipo de mensagem SWIFT usado para transferências internacionais de cliente para cliente (wire transfer). Contém:
- Dados do ordenante (quem envia): nome, conta, banco
- Dados do beneficiário (quem recebe): nome, conta, banco, IBAN (na Europa)
- Valor e moeda
- Data de valor (quando o beneficiário deve receber)
- Código de finalidade da remessa
- Instruções de correspondente

**Fluxo de uma remessa via SWIFT:**
1. Cliente brasileiro solicita remessa de USD 50.000 para fornecedor nos EUA
2. Banco brasileiro gera a mensagem MT103 e a envia via SWIFT ao banco correspondente nos EUA
3. O banco correspondente nos EUA debita a conta nostro do banco brasileiro e credita o banco do beneficiário
4. O banco do beneficiário credita a conta do fornecedor

## Contas nostro e vostro

Para liquidar operações internacionais, os bancos mantêm contas em moeda estrangeira em bancos correspondentes no exterior.

**Conta nostro ("nossa conta"):** conta que o banco brasileiro mantém em um banco estrangeiro, denominada em moeda estrangeira. Exemplo: o Itaú mantém uma conta em dólares no Citibank em Nova York. Quando o Itaú precisa pagar um fornecedor americano, debita sua conta nostro no Citibank.

**Conta vostro ("sua conta"):** é a mesma conta, vista do lado do banco estrangeiro. O Citibank registra a conta do Itaú como uma conta vostro — "a conta de vocês aqui conosco".

**Gestão das contas nostro:**
O banco precisa manter saldo suficiente nas contas nostro para honrar os pagamentos, sem manter saldo excessivo (que rende menos que aplicações). A tesouraria do banco monitora e gerencia esses saldos diariamente.

## Correspondent banking

O modelo de correspondentes bancários é a espinha dorsal do sistema financeiro internacional. Nenhum banco tem presença direta em todos os países — em vez disso, estabelece relacionamentos com bancos locais que atuam como correspondentes.

**Como funciona:**
- Banco brasileiro (banco respondente) estabelece acordo com banco americano (banco correspondente)
- O banco correspondente abre uma conta nostro para o banco brasileiro
- Quando o banco brasileiro precisa pagar em dólares, instrui o correspondente via SWIFT
- O correspondente executa o pagamento localmente

**Cadeia de correspondentes:** em alguns casos, não há correspondente direto entre dois países. A remessa passa por um ou mais bancos intermediários (correspondent chain). Cada intermediário cobra uma taxa e pode atrasar a liquidação.

**Exemplo de cadeia:**
```
Banco brasileiro → Correspondente nos EUA → Banco local no Vietnã → Beneficiário
```

**Riscos do correspondent banking:**
- **Risco de contraparte:** se o correspondente tiver problemas, os fundos podem ficar retidos
- **Risco de compliance:** o banco brasileiro é responsável pelo compliance das operações que passa pelo correspondente — se o correspondente processar operações suspeitas, o banco brasileiro pode ser responsabilizado
- **De-risking:** bancos correspondentes americanos e europeus têm reduzido relacionamentos com bancos de países de alto risco (lavagem de dinheiro, sanções), dificultando o acesso ao sistema financeiro internacional para esses países

**Prazo de liquidação:** remessas via SWIFT geralmente liquidam em 1 a 2 dias úteis para moedas principais (USD, EUR, GBP). Moedas exóticas podem levar mais tempo.
